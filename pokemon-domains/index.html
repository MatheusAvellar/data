Pokémon Domains

This page contains 3 different datasets, all relating to Pokémon and internet domains.

everypokemon.csv (≈12 KB)
A list of every Pokémon name and their respective National Pokedéx numbers, as present in Bulbapedia. As of the last update, this list contains 1008 Pokémon.
pokemondomains.csv (≈14 KB)
A list of every Pokémon name that can become a domain (e.g. Pikachu -‍> pikac.hu). Includes the Pokémon's name, National Dex number, the TLD and the domain itself (e.g. "25, Pikachu, HU, pikac.hu"). As of the last update, this list contains 580 domains.
pokemondomains+status.csv (≈9 KB)
A list of the availability of each Pokémon domain; that is, if a program could detect that they have been purchased. STATUSES MAY NOT BE 100% CORRECT. Domains marked as available are not necessarily purchasable; the list only informs if it could figure out they have been purchased.
As of the last update, there are 43 TAKEN domains, 432 AVAIL domains, and 105 EMPTY responses.

Last updated: 18 November 2022

This dataset is released as public domain. Attribution is appreciated but not required.
