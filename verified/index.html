Verified Accounts on Twitter

This list is a snapshot of verified accounts followed by Twitter Verified on 11 February 2022.

Some accounts followed by Twitter Verified were not verified accounts; those are not included in this dataset.

The list contains 389,705 accounts, and the following information for each one:

id
The Twitter ID of the account; usernames may change, IDs (supposedly) don't
username
The handle (@) of the account
followers
The amount of users that followed the account
following
The amount of users the account followed
likes
The number of tweets the account had liked
tweets
The number of tweets the account had posted
created
The date and time the account was created
protected
If the account is protected (1) or not (0)

You can view/download this dataset here: accounts.csv (≈28 MB).

This dataset is released as public domain. Attribution is appreciated but not required.
